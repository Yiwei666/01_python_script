# 2. tfds-nightly

### 1. `tfds-nightly`概述

`tfds-nightly` 是 `tensorflow-datasets` 的每日构建版本，包含最新的数据集和功能更新。

1. 安装`tfds-nightly`

```
pip install tfds-nightly
```

2. 列出所有数据集名称

```py
import tensorflow_datasets as tfds

# 列出所有可用数据集
for i in  tfds.list_builders():
    print("- ", i)
```
