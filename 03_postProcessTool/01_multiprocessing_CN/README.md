# 1. 项目功能

通过 python 的 multiprocessing 模块多进程加速配位数计算，分别基于 piecewise 和 sigmoid 函数实现

# 2. 文件结构

```
piecewiseCalc.py                                   # 函数模块，仅包含函数
58_sigmoidXYZ轨迹文件配位数多进程计算.py            # 主函数，基于sigmoid计算配位数
58_piecewiseXYZ轨迹文件配位数多进程计算.py          # 基于piecewise阶跃函数计算配位数
```
